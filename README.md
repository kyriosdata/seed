## Como usar (via maven)?

Acrescente a dependência abaixo no arquivo pom.xml:

<pre>
&lt;dependency&gt;
  &lt;groupId&gt;com.github.kyriosdata.seed&lt;/groupId&gt;
  &lt;artifactId&gt;seed&lt;/artifactId&gt;
  &lt;version&gt;1.0.0&lt;/version&gt;
&lt;/dependency&gt;
</pre>
